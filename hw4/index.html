Progressive Web Application

What is a PWA?

A PWA stands for Progressive Web Application. It is a website that works similar to an app. It can be opened in a browser or installed on a device.

The main characteristics of a PWA are that it is user friendly, engaging, responsive, and fast. It can use a service worker so that it works even when offline and can cache important files. It also uses a manifest file to control things like the icon, app name, and display style.
