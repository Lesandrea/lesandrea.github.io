WURFL Device Capabilities

WURFL gives developers access to device capabilities that can be used to identify and test what kind of device is requesting a website or application. Three examples of capabilities are:

Capability Definition Example
built in camerea Shows if the device has a built in camerea Can be used to see if features that allow users to take pictures can be used
brand name It shows the brand of the device, such as Apple or Samsung. It is used to understand what brands are visiting the site and improve testing for those devices.
form factor Identifies the type of device, such as desktop, tablet, or phone. it can be used to change the layout or content depending on whether the visitor is using a phone, tablet, or desktop computer.
